# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities in public GitHub issues.

Use one of these private channels:

1. GitHub Security Advisories (preferred):
   - https://github.com/Olino3/forge/security/advisories
2. Maintainer-coordinated private disclosure channel (if advisory flow is unavailable).

## What to Include

Provide:
- Affected component(s) and version/commit
- Impact assessment
- Reproduction details (private)
- Suggested mitigation (if known)

## Response Expectations

The maintainers will:
- Acknowledge receipt
- Triage severity and impact
- Coordinate remediation and disclosure timeline

## Scope

This policy applies to:
- Forge core workflows and automation
- Plugin and hook infrastructure in this repository
- Security-impacting configuration and release processes

# java-architect Examples

Scenarios showcasing the `java-architect` skill.

---

## Example 1: Microservice Boundary Review

**User Request**: "Evaluate our Java microservices and recommend better service boundaries."

**What the skill does**:
- Maps domain boundaries and coupling hot spots
- Suggests service consolidation or separation by bounded context
- Outputs a phased migration plan

**Output**: `/claudedocs/java-architect_billing-platform_2026-02-12.md`

---

## Example 2: Legacy Monolith Modernization

**User Request**: "Create a modernization strategy for our Java EE monolith."

**What the skill does**:
- Assesses module seams and extraction candidates
- Recommends strangler patterns and integration contracts
- Provides a risk register for each phase

**Output**: `/claudedocs/java-architect_core-monolith_2026-02-12.md`

---

## Example 3: JVM Performance and Scaling

**User Request**: "We need JVM tuning guidance for a high-throughput API."

**What the skill does**:
- Reviews GC strategy, thread pools, and connection management
- Provides tuning experiments and observability checkpoints
- Summarizes scaling limits and remediation steps

**Output**: `/claudedocs/java-architect_api-gateway_2026-02-12.md`

# Technical Writer Memory

This directory stores project-specific documentation patterns and preferences for the @technical-writer agent.

## What's Stored Here

- **Documentation Style Guide**: Project-specific writing style, tone, and formatting preferences
- **Terminology Database**: Preferred technical terms and their definitions
- **Documentation Structure**: Templates and outlines for different document types
- **Common Patterns**: Frequently used code examples, diagrams, and explanations
- **Review Notes**: Feedback and improvements from past documentation reviews

## Usage

The technical writer agent automatically reads from and writes to this directory to:
- Learn project-specific documentation conventions
- Maintain consistency across documentation
- Remember successful documentation patterns
- Adapt to team preferences over time

## Files

Memory files are stored as markdown or JSON for easy version control and human readability.

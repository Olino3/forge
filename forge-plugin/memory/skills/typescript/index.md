# typescript Memory

Project memory for the `typescript` skill. Store compiler configuration, type patterns, and migration status per project.

## Directory Structure

```
typescript/
└── {project-name}/
    ├── project_overview.md
    ├── tsconfig_profile.md
    ├── type_patterns.md
    └── migration_notes.md
```

## Memory Files

### `project_overview.md`

- Project purpose, runtime targets, and key dependencies
- Ownership boundaries and shared type packages

### `tsconfig_profile.md`

- Compiler options, module resolution, and emit targets
- Build tooling and linting rules

### `type_patterns.md`

- Preferred generics, utility types, and shared interfaces
- API contract conventions and validation helpers

### `migration_notes.md`

- Strictness rollout stages and known blockers
- Deprecated patterns and remediation guidance
